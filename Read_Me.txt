# Thank You for getting this 

Step 1: Install the python and pip first , make sure that python environment variable set.

Step 2. check python installed or not . you can check it by running command python --version in cmd

Step 3. : Run the news_scrapper application by double click on it.

Step 4 : It will install the required library and then it will crawl the new from url saved in rss_url.txt file.

Step 5 : After successfully run , a new folder will be created in same directory by name of daily_news and excel file with today date will be saved to this directory
